Neighborhood Map

Neighborhood Map of Chapel Hill, North Carolina. This was also part of Udacit'sy Front-End Nanodegree. For this project, I was required to create a Single Page App (SAP) from scratch using Google Maps API and Knockout.js Framework. The App shows a Google Map interface with a searchbar using Google Maps Autocomplete API and also a list view of all places found in the search. The list is also pre-populated with a few resturants, bars, and cafes.
